# IntelliCamera

An Intelligent with the capability to auto-zoom when detecting objects.

## Features
1. Auto-Zoom When Object Detected.
2. Manual Zoom for users to adjust.
3. Exposure Control
4. Also Provides label with confidence score using a tflite model.
